# MSMEs of Developed India – Landing Page

Landing page for the book **MSMEs of Developed India – An Encyclopedia of MSME Schemes** by CA Mukesh Mohan Gupta.

## Live Website

- **Landing page (live):** [https://msmebook.indiansmechamber.com/](https://msmebook.indiansmechamber.com/)

## About the Book

- **Title:** MSMEs of Developed India – An Encyclopedia of MSME Schemes  
- **Author:** CA Mukesh Mohan Gupta  
- **Edition:** 1st & 2nd Volume, 1st Edition 2026  
- **Pages:** 2008 (2 volumes, 1004 pages each)  
- **Content:** 141 MSME schemes  
- **Format:** Hardcover  

Launched on December 22, 2025 by Sh Piyush Goyal (Hon'ble Minister of Commerce & Industry) and Dr. Rajneesh, IAS (Additional Secretary & Development Commissioner, Ministry of MSME, Government of India).

## Features

- Hero section with book info, highlights, and order CTA  
- Launch event section with photo slideshow  
- Searchable book index (schemes, ministries, stakeholders with page numbers)  
- “Who Should Own this Book” benefits section  
- Order via Razorpay  
- Responsive layout and accessibility-friendly markup  

## Tech Stack

- HTML5  
- CSS3 (custom styles in `app.css`)  
- Vanilla JavaScript  
- Google Fonts (Lora)
